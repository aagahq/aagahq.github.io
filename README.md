## How to Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g., `aaga-portfolio`).
2. Initialize git in this directory:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
3. Add your GitHub repository as a remote:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/aaga-portfolio.git
   ```
4. Push your code:
   ```bash
   git push -u origin main
   ```
5. Go to your repository settings on GitHub, navigate to **Pages**, and set the source branch to `main`.
6. Your website will be live at `https://YOUR_USERNAME.github.io/aaga-portfolio/`.
