# Kali Linux Tools Overview

This directory contains summaries of key security tools available on Kali Linux. These summaries are intended for quick reference and to understand the capabilities of each tool.

## Key Tools
- [Nmap](./nmap.md): Network scanning and discovery.
- [Metasploit](./metasploit.md): Exploitation and post-exploitation.
- [TShark](./tshark.md): Network protocol analysis.
- [SQLMap](./sqlmap.md): SQL injection testing.
- [John the Ripper](./john.md): Password cracking.
- [Hydra](./hydra.md): Login cracking.
- [Aircrack-ng](./aircrack-ng.md): WiFi network security auditing.
- [Burp Suite](./burpsuite.md): Web application security testing.
- [Responder](./responder.md): LLMNR, NBT-NS and MDNS poisoner.
- [NetExec](./netexec.md): Network service exploitation tool.
- [DIRB](./dirb.md): Web content scanner.
- [dnsenum](./dnsenum.md): DNS enumeration tool.
- [Ettercap](./ettercap.md): Man-in-the-middle attack tool.
- [Nikto](./nikto.md): Web server scanner.
- [Gobuster](./gobuster.md): Directory/file and DNS enumeration.
- [WPScan](./wpscan.md): WordPress security scanner.
- [Hashcat](./hashcat.md): Advanced password recovery.
- [WhatWeb](./whatweb.md): Website identification.
- [WafW00f](./wafw00f.md): WAF identification.
- [tcpdump](./tcpdump.md): Network protocol analyzer.
- [enum4linux](./enum4linux.md): Windows/Samba enumeration.
- [dnsrecon](./dnsrecon.md): DNS enumeration tool.
- [theHarvester](./theharvester.md): OSINT gathering tool.
- [Medusa](./medusa.md): Parallel login brute-forcer.
- [NBTScan](./nbtscan.md): NetBIOS network scanner.
- [Reaver](./reaver.md): WPS brute force tool.

These summaries cover only a fraction of the tools available in Kali. For more information, refer to the individual tool's documentation or man pages.
